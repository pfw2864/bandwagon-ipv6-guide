### 第二步：确认你的机房支持 IPv6

买完之后登录后台，进入 Services → My Services，找到自己的 VPS，点 KiwiVM Control Panel 进入管理面板。

左侧菜单找一下机房信息，看当前机房是否在上面列出的支持列表里。如果不在，可以用 KiwiVM 的 Migrate 功能免费迁移到支持 IPv6 的机房。CN2 GIA-E 套餐可切换的机房超过 12 个，基本不用担心没地方迁。

### 第三步：KiwiVM 控制面板激活 IPv6

这一步很简单，按顺序来：

1. 进入 KiwiVM 控制面板左侧菜单，找到 **IPv6 network**
2. 点击进入后，界面会显示 **IPv6 Management** 页面
3. 点击 **Activate /64 IPv6 subnet**，系统会自动分配一个 /64 的 IPv6 地址段（也就是说地址数量多到基本用不完）
4. **关键步骤**：激活之后必须重启 VPS 才能生效。在 KiwiVM 主界面点 Stop 再 Start，或者用 Restart 按钮都行
5. VPS 重启完成后，SSH 登录进去，运行 `ifconfig`，找到 inet6 那一行，就是分配到的 IPv6 地址

验证一下：用 `ping6 google.com` 测试是否能正常 ping 通，通了就说明 IPv6 配置成功。

## 常见问题

**Q：KVM 套餐和 CN2 GIA-E 套餐 IPv6 有什么区别？**

都能用，IPv6 本身没有线路差异。区别在于 IPv4 那条线——KVM 套餐走普通骨干线路，晚高峰容易拥堵；CN2 GIA-E 三网走 CN2 GIA 优化线路，延迟稳、丢包少。如果你主要用 IPv6 访问国外服务，买 KVM 套餐就够；如果还需要用 IPv4 做国内外双向访问，建议选 CN2 GIA-E。

**Q：激活 IPv6 后原来的 IPv4 地址还在吗？**

在。激活 IPv6 是额外加一个网络接口，不影响原有的独立 IPv4 地址。一台 VPS 同时有 IPv4 + /64 IPv6 段。

**Q：搬瓦工 IPv6 是免费的还是额外付费的？**

免费包含在套餐里。买套餐、激活、使用，全程不额外收费。

**Q：香港/日本套餐用户想要 IPv6 怎么办？**

目前这两类高端套餐还不支持原生 IPv6。如果确实需要 IPv6，可以暂时用隧道方案解决，或者等待官方扩展支持范围——从历史来看，搬瓦工的 IPv6 支持一直在扩展，早期只有 DC99，现在洛杉矶系列基本都上了。

**Q：激活之后 ping6 不通，怎么处理？**

先试试重启 VPS。不行的话，手动检查 `/etc/network/interfaces` 这个配置文件，确认 IPv6 网络接口的配置是否正确写进去了。Ubuntu 用户还可以查一下 netplan 配置。
